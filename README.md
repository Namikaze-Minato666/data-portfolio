# 张志豪数据分析作品集网站

这是一个用于求职展示的个人数据分析作品集网站，面向数据分析实习、商业分析实习、数据运营实习和 BI 分析相关岗位。

网站使用纯 HTML、CSS 和少量 JavaScript 编写，不依赖 React、Vue、Next.js 或任何需要安装的框架。直接双击 `index.html` 即可在浏览器中预览。

## 页面内容

- 首页 Hero：个人定位、求职方向、GitHub 和联系方式入口
- 项目展示：Instacart 用户复购分析、Olist 电商经营分析、酒店预订取消行为分析
- 技能区：SQL、Python 数据分析、BI 可视化、业务分析方法和常用工具
- 关于我：学习背景与项目实践方向
- 联系方式：手机与邮箱

## GitHub Pages 部署方法

1. 将本项目推送到 GitHub 仓库。
2. 进入仓库页面，点击 `Settings`。
3. 在左侧菜单中找到 `Pages`。
4. 在 `Build and deployment` 中选择：
   - Source：`Deploy from a branch`
   - Branch：`main`
   - Folder：`/root`
5. 点击 `Save`。
6. 等待 GitHub Pages 构建完成后，即可通过页面给出的链接访问作品集网站。
