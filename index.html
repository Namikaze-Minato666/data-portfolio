Market Basket 01

Instacart 用户复购行为与购物篮分析项目

MySQL、CTE、窗口函数、中间表、Excel、Tableau

基于 Instacart Market Basket Analysis 数据集，围绕用户购买行为、购物篮规模、复购规律、商品关联规则和用户分层展开分析。项目处理 3,421,083 条订单记录和 32,434,489 条历史订单商品明细记录。

  • 构建用户级、订单级和商品级中间汇总表，降低 3000 万级明细表重复扫描成本。
  • 计算整体复购率、购买间隔复购率、订单阶段复购率。
  • 构建严格递进的用户行为漏斗。
  • 使用 support、confidence、lift 分析商品关联规则。
  • 基于购买频率和复购率完成用户分层。

E-Commerce 02

Olist 巴西电商平台经营分析项目

MySQL、窗口函数、RFM、Excel、Tableau

基于 Olist Brazilian E-Commerce Public Dataset，围绕平台整体经营表现、销售趋势、商品品类、物流履约、用户评分、卖家表现、支付方式和 RFM 用户分层展开分析。

  • 构建 GMV、AOV、取消率、平均评分、延迟率等核心经营指标。
  • 分析 2017-2018 年月度订单量和 GMV 趋势。
  • 从 GMV、订单量、AOV 和评分维度识别核心品类。
  • 分析物流延迟与用户评分关系。
  • 使用 RFM 模型识别普通用户、流失用户、新客高消费和高价值用户。

Python EDA 03

酒店预订取消行为分析项目

Python、Pandas、NumPy、Matplotlib、Seaborn、Jupyter Notebook

基于酒店预订数据集，围绕订单取消行为展开分析，完成原始数据读取、缺失值处理、特征工程、取消率分组统计和可视化分析。

  • 使用 Pandas 处理 119,390 条酒店预订数据。
  • 针对 company、agent、country、children 等字段设计差异化缺失值处理策略。
  • 构造总入住人数、总入住晚数、是否有儿童、是否通过代理预订等特征。
  • 从酒店类型、提前预订天数、客户类型、市场来源、押金类型等维度分析取消率差异。
  • 使用 Matplotlib / Seaborn 输出可视化图表。
